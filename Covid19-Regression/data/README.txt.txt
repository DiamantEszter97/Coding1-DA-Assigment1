Data Analyis 2 - Coding 1: Linear Regression


The data was dowloaded on 2nd of November, 2020
The attached dataset contains data on countries, population, the number of confirmed covid-19 cases in the country, the number of death resulted from covid-19 infection, the number of recovered from covid-19 infection cases, and the number of currently active covid-198 cases.

The main of the analysis to identify a linear regression between the number of confirmed ases and number of death resulted from the covid-19 infection.

The data was collected from the following sources:

data on countries and population: WDI package in R
data on covid-19 cases: 'https://raw.githubusercontent.com/CSSEGISandData/COVID-19/master/csse_covid_19_data/csse_covid_19_daily_reports/
